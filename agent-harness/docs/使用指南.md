# 使用指南

## 启动 Harness

### Windows (PowerShell)
```powershell
.\scripts\run-tests.ps1
```

### Linux/macOS (Bash)
```bash
./scripts/run-tests.sh
```

## 在 Obsidian 中工作

### 创建新测试
1. 在 `tests/` 下创建 Markdown 文件
2. 使用模板：`Ctrl+P` → "Templates: Insert template"
3. 运行测试

### 查看结果
测试结果保存在 `results/run_YYYYMMDD_HHMMSS/` 中。

### 使用双向链接
- `[[` 创建链接到其他笔记
- `Ctrl+点击` 跳转到链接
- 查看关系图谱：`Ctrl+G`

## 配置 Agent

编辑 `config/opencode.json`：

```json
{
  "model": "moonshot/kimi-k2.6",
  "agent": {
    "your-agent": {
      "prompt": "你的系统提示词",
      "tools": ["read", "edit", "bash"]
    }
  }
}
```

## 添加 Skills

Skills 安装在 `~/.config/opencode/skills/`，会自动加载。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+G` | 关系图谱 |
| `Ctrl+P` | 命令面板 |
| `Ctrl+S` | 保存文件 |
| `Ctrl+点击` | 打开链接 |
