# 项目结构

```
agent-harness/
├── .obsidian/              # Obsidian 配置
│   ├── app.json            # 应用设置
│   ├── appearance.json     # 外观设置
│   ├── core-plugins.json   # 启用的插件
│   ├── hotkeys.json        # 快捷键
│   ├── plugins/            # 插件目录
│   ├── snippets/           # CSS 片段
│   └── themes/             # 主题
├── config/                 # Agent 配置
│   └── opencode.json       # 主配置
├── tests/                  # 测试用例
│   ├── unit/              # 单元测试
│   ├── integration/       # 集成测试
│   └── benchmarks/        # 基准测试
├── agents/                 # Agent 定义
├── fixtures/               # 测试数据
│   ├── code-samples/      # 代码示例
│   └── mock-data/         # Mock 数据
├── scripts/                # 运行脚本
│   ├── run-tests.sh       # Bash 脚本
│   └── run-tests.ps1      # PowerShell 脚本
├── results/                # 测试结果
│   └── reports/           # 报告
├── docs/                   # 文档
│   ├── Home.md            # 主页
│   ├── 测试目录.md        # 测试索引
│   └── 使用指南.md        # 使用说明
├── templates/              # 模板
│   ├── test-template.md   # 测试模板
│   └── agent-definition.md # Agent 模板
└── README.md               # 项目说明
```

## 关键文件

| 文件 | 用途 |
|------|------|
| `config/opencode.json` | Agent 配置、模型设置 |
| `scripts/run-tests.ps1` | Windows 测试运行器 |
| `scripts/run-tests.sh` | Linux/macOS 测试运行器 |
| `.obsidian/app.json` | Obsidian 应用设置 |

## 数据流

```
Tests (.md) → Agent (opencode) → Results (.txt)
     ↑                                 ↓
Templates ←---------------------- Reports (.md)
```
