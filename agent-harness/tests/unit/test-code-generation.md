# Test Prompt: Basic Code Generation

## Task
Write a Python function that calculates the factorial of a number using recursion.

## Requirements
- Function name: `factorial(n)`
- Input: non-negative integer
- Output: factorial value
- Handle edge cases (0, negative numbers)
- Include docstring

## Expected Behavior
- factorial(0) = 1
- factorial(5) = 120
- factorial(-1) should raise ValueError

## Evaluation Criteria
- [ ] Function compiles without errors
- [ ] Correct implementation of recursion
- [ ] Proper error handling
- [ ] Code style and documentation
- [ ] All test cases pass
